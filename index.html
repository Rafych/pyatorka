About

Pyatorka overlays two DXF drawings and highlights exactly where they differ. Load File A and File B, adjust colors and opacity, and any part that doesn't perfectly coincide — lines, curves, circles, points, or text — is marked in a separate color for each file.

It's a small, self-contained desktop app built for a single job: comparing two CAD drawings at a glance.

О приложении

Пятёрка накладывает два DXF-чертежа друг на друга и подсвечивает именно те места, где они отличаются. Загрузите Файл A и Файл B, настройте цвета и прозрачность — и всё, что не совпадает полностью (линии, кривые, окружности, точки или текст), будет выделено своим цветом для каждого файла.

Это компактное автономное настольное приложение, созданное для одной задачи: быстро сравнивать два CAD-чертежа.

概要

ピャトルカは2つのDXF図面を重ね合わせ、異なる箇所だけを正確にハイライト表示するツールです。ファイルAとファイルBを読み込み、色や不透明度を調整すると、線・曲線・円・点・文字のうち完全に一致しない部分だけが、ファイルごとに異なる色で表示されます。

2つのCAD図面をひと目で比較する、という一つの目的のために作られた、単体で動くシンプルなデスクトップアプリです。
